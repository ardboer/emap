### Lazy Loading State Machine

```mermaid
stateDiagram-v2
    [*] --> Idle: Component Mount
    Idle --> Approaching: Distance < 250px
    Approaching --> Loading: shouldLoad = true
    Loading --> Loaded: Ad Ready
    Loading --> Failed: Load Error
    Loaded --> InViewport: User Scrolls
    InViewport --> Viewed: Track Impression
    Viewed --> Idle: Scroll Away
    Failed --> Idle: Reset

    note right of Approaching
        Threshold: 250px
        One at a time
    end note

    note right of InViewport
        Track viewability
        Log analytics
    end note
```

### Data Flow Sequence

```mermaid
sequenceDiagram
    participant U as User
    participant S as ScrollView
    participant D as DisplayAd
    participant H as useInView Hook
    participant M as LazyLoadManager
    participant B as BannerAd
    participant A as Analytics

    U->>S: Scrolls Down
    S->>D: Render Component
    D->>H: Monitor Position
    H->>H: Calculate Distance

    alt Distance < 250px
        H->>D: onEnterView()
        D->>M: startLoading(adId)
        M->>A: Log 'approaching'
        D->>D: setShouldLoad(true)
        D->>B: Render with shouldLoad
        B->>B: Load Ad
        B->>M: markAsLoaded()
        M->>A: Log 'loaded'
    end

    alt Ad Enters Viewport
        H->>M: markAsViewed(adId)
        M->>A: Log 'viewed'
        M->>A: Track Viewability
    end

    U->>S: Scrolls Away
    H->>D: onExitView()
    D->>M: Update State
```

## Component Architecture

### DisplayAd Component Structure

```typescript
DisplayAd
├── useInView Hook
│   ├── Position Tracking
│   ├── Threshold Detection (250px)
│   └── Callbacks (onEnter/onExit)
├── State Management
│   ├── shouldLoad: boolean
│   ├── isLoaded: boolean
│   └── hasError: boolean
├── LazyLoadManager Integration
│   ├── Register Ad
│   ├── Track Loading
│   └── Track Viewing
└── Rendering Logic
    ├── Placeholder (if not loaded)
    ├── BannerAd (if shouldLoad)
    └── Error State (if failed)
```

### useInView Hook Architecture

```typescript
useInView
├── Layout Measurement
│   ├── onLayout callback
│   ├── Component dimensions
│   └── Position in scroll
├── Scroll Tracking
│   ├── ScrollView context
│   ├── Current scroll position
│   └── Viewport bounds
├── Intersection Calculation
│   ├── Distance from viewport
│   ├── Threshold comparison
│   └── Visibility percentage
└── State & Callbacks
    ├── inView: boolean
    ├── onEnterView()
    └── onExitView()
```

### displayAdLazyLoadManager Service

```typescript
displayAdLazyLoadManager
├── Ad Registry
│   ├── Map<adId, AdState>
│   ├── Loading states
│   └── Timing data
├── Lifecycle Management
│   ├── registerAd()
│   ├── startLoading()
│   ├── markAsLoaded()
│   ├── markAsFailed()
│   ├── markAsViewed()
│   └── unregisterAd()
├── State Queries
│   ├── isLoading()
│   ├── isLoaded()
│   └── shouldLoad()
└── Analytics Integration
    ├── Track events
    ├── Calculate metrics
    └── Report statistics
```

## Integration Points

### Article Detail Integration

```mermaid
graph LR
    A[Article Screen] --> B[After Lead Text]
    A --> C[RichContentRenderer]
    B --> D[DisplayAd - Lazy]
    C --> E[Content Paragraphs]
    E --> F[DisplayAd - Lazy]
    E --> G[DisplayAd - Lazy]

    style D fill:#e1f5ff
    style F fill:#e1f5ff
    style G fill:#e1f5ff
```

**Implementation:**

- After-lead ad: Single DisplayAd with lazy loading
- In-content ads: Multiple DisplayAds injected between paragraphs
- Each ad independently tracks its viewport position
- Sequential loading as user scrolls

### List View Integration

```mermaid
graph TB
    A[FlatList/ScrollView] --> B[Block 1]
    A --> C[Block 2]
    A --> D[DisplayAd - Lazy]
    A --> E[Block 3]
    A --> F[Block 4]
    A --> G[DisplayAd - Lazy]

    style D fill:#e1f5ff
    style G fill:#e1f5ff
```

**Implementation:**

- Ads inserted between content blocks
- FlatList optimization + lazy loading
- Respects `windowSize` for performance
- Works with `onViewableItemsChanged`

## Performance Considerations

### Memory Management

```
Before Lazy Loading:
┌─────────────────────────────────┐
│ All Ads Loaded Immediately     │
│ Memory: ~15-20MB per ad         │
│ Total: 60-100MB for 4-5 ads    │
└─────────────────────────────────┘

After Lazy Loading:
┌─────────────────────────────────┐
│ Only Visible + Approaching Ads  │
│ Memory: ~15-20MB per ad         │
│ Total: 30-40MB for 1-2 ads     │
│ Savings: ~50-60% reduction      │
└─────────────────────────────────┘
```
